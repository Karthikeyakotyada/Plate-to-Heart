# 🍽️ Plate to Heart

Plate to Heart is a web-based food donation platform designed to reduce food wastage by connecting food donors such as hotels, hostels, and restaurants with charities, NGOs, and volunteers who can distribute surplus food to people in need.

## 🌟 Features

### 🏠 Dashboard

* View total available plates
* Track active donors
* Monitor pickup history
* Quick access to important actions

### ⬆️ Upload Food Availability

* Add available food plates
* Select donor type (Hotel, Hostel, Charity)
* Specify food quantity
* Add food type (Veg / Non-Veg)
* Add curry or food details
* Capture location for easy pickup

### 📦 Available Plates

* View all active food donations
* See donor information
* Check available plate count
* Request food pickups

### 📜 Pickup History

* Track completed pickups
* View pickup details
* Monitor food distribution records

### 📍 Location Support

* Automatic GPS location detection
* Address retrieval using geolocation
* Improved pickup coordination

### ☁️ Firebase Integration

* Real-time database updates
* Cloud data storage
* Instant synchronization across users

## 🎯 Purpose

Every day, large amounts of edible food are wasted while many people struggle to access meals. Plate to Heart aims to bridge this gap by creating a simple and efficient platform where surplus food can quickly reach those who need it most.
